🏠 全体の構成（走りポーズ版）

  • 横 18マス（腕の広がり含む）× 高さ 15段 × 奥行き 8マス
  • 耳（行1〜2）= 腰壁 = 2単位。左右に大きく広がるフロッピーイヤー
  • 顔（行3〜8）= 普通の壁+低い壁 = 6単位。白+黒（目・鼻）
  • 胴体（行9〜13）= 普通の壁 = 5単位。赤い衣装+窓+ドア。建物本体（幅12マス）
  • 脚（行14〜15）= 腰壁 = 2単位。走りポーズで左右非対称に開いている
  • 合計 2+6+5+2 = 15単位 = 高さ上限ぴったり
  • 床面積: 12×8 = 96タイル。腕と脚は壁のみ（床なし）
🏃 公式画像（list-pochacco.png）の走りポーズを再現:
・腕: 左腕が前方に大きく伸び（行9-10左側）、右腕は後方（行9-10右側）
・脚: 左脚が前に出て（行14-15左）、右脚が後ろ（行14-15右）
・耳: 走りの勢いでバサッと横に広がっている
・全体に非対称なのが走りポーズの特徴。左右で腕・脚の位置が違うよ
⚠️ 壁1枚 = 1色ルール: 1枚の壁に1色しか塗れない。 色を変えたい行は壁を分ける必要があるよ。
→ 顔は白と黒が行ごとに混在するから、低い壁を1行ずつ積むのがおすすめ。 詳しくは「顔」「建物」ボタンで確認してね。

👂 耳の作り方（最重要パーツ！）

  • 走りの勢いでバサッと横に広がったフロッピーイヤー
  • 左耳: 列2-3 → 列3-4（行1→行2で斜めに下がる）
  • 右耳: 列16-17 → 列15-16（行1→行2で斜めに下がる）
  • 行3では耳先端（列4と列15）だけが残り、顔の丸みと繋がる
  • 壁の組み方: 行1〜2は腰壁 = 2単位。黒1色でOK
  • 色: 全部黒。シンプルに腰壁を黒で塗るだけ

😊 顔の作り方

  • 行3〜8の6行 = 普通の壁+低い壁の高さ
  • 行3: 耳の先端が顔の横に残る＋額（白）
  • 目: 行5の列7 と 列12（1マスの黒）← 公式の小さなつぶらな目
  • 鼻: 行7の列9-10（横長2マスの黒）← 公式の楕円鼻
  • 目と鼻の間に1行のスペース（行6）← ゆったりした配置を再現
  • 行8: あご（下すぼまりの白）
⚠️ 塗り分け: 行5（目）と行7（鼻）に黒が混じる。
→ おすすめ: 低い壁6枚を1行ずつ積んで、各行を個別にペイント

🧱 胴体＆脚の作り方（建物部分）

  • 胴体（行9〜13）= 普通の壁 = 5単位。赤い衣装。幅12マス（列4〜15）
  • 行9-10: 腕が左右に伸びている（走りポーズ！白い手）
  • 窓: 行11。左右に水色ガラス2枚ずつ + グレー窓枠
  • ドア: 行12-13の中央。黒フレーム + ミントガラス（4マス幅）
  • 脚（行14〜15）= 腰壁 = 2単位。走りで前後に開いた白い脚
⚠️ 塗り分け:
→ 行9-10: 赤+白（腕）が混在 → 低い壁×2で1行ずつ
→ 行11: 窓あり → 低い壁1枚
→ 行12-13: ドアあり → 腰壁1枚
→ 行14-15: 白い脚 → 腰壁1枚
= 低い壁3 + 腰壁2 = 3+4 = 7単位... ⚠️ 実際は行9-13が5行、行14-15が2行なので普通の壁1枚(5)+腰壁1枚(2) = 7単位 ✅

🔨 作る順番

1
脚レベル（行14-15）の壁 地面に腰壁を白で配置。走りポーズの脚を再現（非対称！）
2
胴体の床＋壁（行9-13） 12×8の床を敷き、普通の壁で四方を囲む → 赤で塗る
3
腕を作る（行9-10の左右） 建物の両サイドに白い壁を伸ばす。走りポーズなので非対称に！
4
窓とドアを配置 行11に窓、行12-13にドアを設計図通りに
5
2階の床（＝胴体の天井） 壁の上に床パーツ。顔の土台になる
6
顔の壁（行3-8） 低い壁6枚を1行ずつ。白で塗って、目と鼻だけ黒に
7
耳を作る（行1-2） 床を挟んで腰壁を追加 → 黒く塗る。左右に大きく広げて！
8
庭を飾って完成！ 紫陽花＋低木を正面に。花壇で統一感
⚠️ 壁を積むときの注意: 壁の上に壁を直接置けない！ 必ず間に床パーツを挟む（床→壁→床→壁→…の順）。 床パーツは天井と兼用になるから無駄にならないよ。
